# 智能文件清理工具

一个简单易用的文件清理工具，支持批量删除指定类型的文件。

## 功能特点

- 支持拖拽文件夹
- 预设常用文件类型（.lrc, .txt, .log, .tmp）
- 支持自定义文件类型
- 支持单层/多层文件夹模式
- 实时显示操作日志
- 美观的图形界面
- 支持多线程操作

## 使用方法

1. 双击运行 `智能文件清理工具.exe`
2. 选择要清理的文件夹：
   - 点击"浏览"按钮选择文件夹
   - 或直接将文件夹拖放到输入框中
3. 选择要删除的文件类型：
   - 点击预设类型按钮（.lrc, .txt, .log, .tmp）
   - 或在自定义类型输入框中输入其他类型（用逗号分隔，例如：.bak, .old）
4. 选择文件夹模式：
   - 单层模式：仅处理指定文件夹中的文件
   - 多层模式：处理指定文件夹及其所有子文件夹中的文件
5. 点击"开始删除"按钮
6. 在日志区域查看删除进度和结果

## 注意事项

1. 程序运行需要Windows系统
2. 删除操作不可恢复，请谨慎操作
3. 建议在删除前备份重要文件
4. 如果遇到杀毒软件报警，这是误报，可以放心使用
5. 使用多层模式时请特别注意，确保不会误删重要文件
